#  Checkpoint 6 – Python y APIs

##  Descripción

Este proyecto contiene la documentación y un ejercicio práctico sobre conceptos fundamentales de Python y APIs.

El objetivo es explicar de forma clara y sencilla temas clave para principiantes en el desarrollo.

---

##  Contenidos

* Clases en Python
* Método `__init__`
* Verbos de API (GET, POST, PUT)
* MongoDB (NoSQL)
* Qué es una API
* Qué es Postman
* Polimorfismo
* Métodos dunder
* Decoradores

---

##  Ejercicio práctico

Se ha creado una clase `Usuario` que:

* Usa el método `__init__`
* Guarda nombre de usuario y contraseña
* Crea un objeto
* Muestra los datos
